# Blockchain Chat Application (ChatBuddy)

A decentralized Web3 Chat Application built using Next.js, Solidity, Hardhat, Ethers.js, and MetaMask.
This DApp enables users to connect their wallet and send secure on-chain messages using the Ethereum blockchain.
